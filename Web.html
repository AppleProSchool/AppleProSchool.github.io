Html

HTML is short for HyperText Markup Language, HTML defines the structure and layout of a Web document by using a variety of tags and attributes.

Css

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen. It can control the layout of multiple web pages all at once.

JavaScript

Javascript (JS) is a scripting language. It is used to enhance HTML pages and is commonly found embedded in HTML code. JavaScript renders web pages in an interactive and dynamic fashion.
